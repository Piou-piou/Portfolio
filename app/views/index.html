Mes compétences

“ Ici le possible est déjà fait, l'impossible est en cours et pour le miracle il me faut un délai de 48h ! “

PHP7 est le langage que j’utilise côté serveur afin de rendre les sites dynamiques.

MySQL est le moteur de base de données dont je me sers pour administrer les données des sites

Wordpress est le CMS que je mets en place lors d’un besoin simple d’administration de site ou blog

Ribs est un framework que j’ai développé et que j’utilise lors de création d’applications web

HTML5 est le langage de balises qui me permet de mettre en place la structure et prévoir le SEO

CSS3 est un langage en cascade qui me permet d’intégrer le design de sites et aplications web

SASS est un préprocesseur qui me permet de rendre le CSS plus maintenable et plus organisé

JS et Jquery sont des langages que j’utilise pour mettre en place diverses animations.

Bootstrap est un framework qui permet de réaliser des mises en page adaptées sur mobile

Foundation est comme bootstrap, je l’utilise pour Ribs et pour des applications

Twig est un moteur de template qui me permet d’injecter des variable dans du code HTML

Grunt est un outils qui me permet d'automatiser la compilation des fichier sass en css et l'envoi en FTP sur les serveurs.
